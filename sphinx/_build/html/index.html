About AftFore¶

AftFore is a python module for generating aftershock forecasts based on the method proposed in Omi et al., (2013). Especially AftFore aims to improve the forecasts at the early stage (e.g. from the first few hours to the few days after the main shock).

Anyone can freely use this code without permission for non-commercial purposes, and any feedback is welcome. If you present results obtaind by using AftFore, please appropriately cite Omi et al., (2013).

Acknowledgement¶

This package contains a sample file of the 1995 Kobe aftershock sequence from the JMA unified catalog (http://www.data.jma.go.jp/svd/eqev/data/bulletin/hypo_e.html). I appreciate JMA for providing the data.

Reference¶

T. Omi, Y. Ogata, Y. Hirata, and K. Aihara, “Forecasting large aftershocks within one day after the main shock”, Scientific Reports 3, 2218 (2013).
http://dx.doi.org/10.1038/srep02218
